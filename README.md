# rules-us-or

Oregon rules in Akoma Ntoso XML format.

## Structure

```
rules-us-or/
├── statutes/      # Oregon Revised Statutes (ORS)
└── regulations/   # Oregon Administrative Rules (OAR)
```

## Format

All documents are encoded in [Akoma Ntoso](http://www.akomantoso.org/) XML format, an OASIS standard for legislative and legal documents.

## Sources

- **Oregon Revised Statutes (ORS)**: https://www.oregonlegislature.gov/bills_laws/Pages/ORS.aspx
- **Oregon Administrative Rules (OAR)**: https://sos.oregon.gov/archives/Pages/oregon_administrative_rules.aspx
